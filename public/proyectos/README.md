# Imágenes de Proyectos — TodoCasa Construcciones

## Estructura

```
proyectos/
├── ampliaciones/
│   ├── ampliacion-01a.jpg
│   ├── ampliacion-01b.jpg
│   └── ...
├── quinchos/
│   ├── quincho-01a.jpg
│   └── ...
├── terrazas/
│   ├── terraza-01a.jpg
│   └── ...
└── piscinas/
    ├── piscina-01a.jpg
    └── ...
```

## Convenciones

- **Tamaño:** 1200x900px (aspect ratio 4:3)
- **Formato:** JPG, calidad 80-85%
- **Peso máximo:** 200KB por imagen
- **Naming:** `{tipo}-{numero}{letra}.jpg`
  - `a` = vista general, `b` = detalle, `c` = lateral, `d` = terminado

## Optimización recomendada

- [Squoosh](https://squoosh.app/)
- [TinyPNG](https://tinypng.com/)
