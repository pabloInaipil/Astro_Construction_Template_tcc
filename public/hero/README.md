# Imágenes y Video del Hero — TodoCasa Construcciones

## Archivos necesarios

```
hero/
├── slide-01.jpg    ← Slide principal (TodoCasa Construcciones)
├── slide-02.jpg    ← Terrazas
├── slide-03.jpg    ← Quinchos
├── slide-04.jpg    ← Ampliaciones
├── slide-05.jpg    ← Piscinas
└── video-fondo.mp4 ← Video de fondo (opcional)
```

## Especificaciones

### Imágenes
- **Tamaño:** 1920x1080px mínimo (fullscreen hero)
- **Formato:** JPG, calidad 80-85%
- **Peso máximo:** 300KB por imagen (optimizar con Squoosh)
- **Aspecto:** Horizontal, preferir fotos con zona central oscura para legibilidad del texto

### Video (opcional)
- **Formato:** MP4 (H.264) — máxima compatibilidad
- **Duración:** 10-20 segundos, en loop
- **Resolución:** 1920x1080 o 1280x720
- **Peso ideal:** < 5MB
- **Tips:** Sin audio, movimientos lentos, desaturado levemente

Si no se agrega video, el slider de imágenes funciona solo.
